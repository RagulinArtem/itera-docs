4 техники — как можно писать

Разные форматы для разных целей: от свободного письма до структурированной работы с тревогой.

📝
Свободное письмо
Любые мысли, без структуры. Для выгрузки эмоций и первого шага.
🧠
СМЭР (КПТ)
Разбор ситуации по шагам: что случилось → что подумал → что почувствовал → как отреагировал.
💗
Работа с тревогой
Декатастрофизация: проверка тревожной мысли через худший/лучший/реалистичный исход.
🌟
Благодарность
Три вещи, за которые благодарны сегодня. Тренирует внимание к хорошему.
Как работает ИИ

Главная магия приложения: от момента, когда вы нажали «сохранить», до тёплого отклика.

✍️

⚡
Моментальный отклик
После каждой записи — 2-3 тёплых предложения. Замечает главное, задаёт мягкий вопрос.

Итоги недели
Автоматический инсайт по 7 дням: ключевая тема, источник опоры, рекомендация.

Отчёты
Глубокий анализ за месяц или всё время. Темы, эмоциональный спектр, паттерны.
🔍
Вопросы к записям
Спросите «что вызывает тревогу?» — ИИ ответит по вашим же записям с цитатами.
Под капотом — технологии

Всё работает на надёжных, проверенных компонентах. Данные защищены, ИИ — от OpenAI.

⚛️
Интерфейс
React + Tailwind
Быстрый, адаптивный, работает на мобильных и десктопе.
⚡
Сервер
FastAPI (Python)
Принимает запросы, общается с ИИ, сохраняет записи.
🗄️
База данных
Supabase (PostgreSQL)
Хранит профили и записи. Row-Level Security — записи видит только их владелец.
🤖
ИИ
OpenRouter → GPT-4o
Быстрая модель для откликов, мощная — для отчётов.
🏗️
Хостинг
Docker + VPS
Живёт на собственном сервере по адресу itera-diary.ru.
🔒
Приватность
End-to-end
HTTPS, изолированные аккаунты, экспорт и удаление данных — по кнопке.
